## 作品概要
Webサイト制作の基礎スキルの習得を目的として制作した、架空の運送会社のコーポレートサイト（トップページのみ）です。  
実務を想定した制作経験を得るため、無料教材ではなく、プロのデザイナーが制作したデザインカンプを購入してコーディングしました。

## 工夫ポイント
+ **CSS設計**：  
  色や余白などの共通値をCSS変数で管理し、保守性の高いコードを意識しました。  
  また、役割が理解できるクラス名にすることで、可読性の向上を図りました。  

+ **CSSとJsの責務分離**：  
  スタイル指定はCSSで管理し、JavaScriptではクラスの付与や削除といった制御のみを行うことで、役割を明確に分けました。  
  これにより、意図しない動作を防ぎ、保守性の高いコードを実現しています。  

+ **Observer監視設計**  
  IntersectionObserverは1つにまとめ、条件分岐で処理タイミングを分ける設計にしました。  
  これにより、冗長化を防ぎ、意図しない動作が起きにくいコードを実現しました。

## 改善点・今後の目標
+ IntersectionObserverなど、初めて扱う技術の理解に時間がかかり、結果として制作期間が想定より長くなりました。  
  今回の制作で学んだ技術を整理し、テンプレートとして蓄積することで、次回以降の制作では作業時間の短縮を図りたいと考えています。  

+ 今回はフロントエンドの基礎スキルの習得を重視して制作を行ったため、GSAPやパララックスなどを用いた応用的な表現に取り組めませんでした。  
  次回の制作では、より高度な表現技術も取り入れ、ユーザー体験の向上に貢献できるサイトの制作を目指していきます。
